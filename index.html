Eco-Solve

Learn more about the environment and make more sustainable choices!

Calling all environmentalists!
Eco-Solve is an environmental education website where users can learn more about the environment, current environmental issues, and make more sustainable choices.
To do this, users can watch videos, take quizzes, play games, and even set their own sustainability goals.

Videos

The videos page of Eco-Solve is for all visual learners wanting to learn more about the environment.
Currently, the only video available is the award-winning short film "Echoes of Humanity",
a dystopian tale about the last human alive leaving a message about how humanity destroyed the environment.

Quizzes

 Want to test yourself?
Eco-Solve's quiz webpage is sure to provide both a challenge and a benchmark for how well you are progressing.
The quizzes are 5-10 questions long and cover topics about environmental science.
The questions are in a random order, so be prepared for whatever comes your way.

Mini-Game

 For some downtime and fun, Eco-Solve offers a mini-game called Eco-Quest.
In the game, there is a tank which collects greenhouse gas particles, and the more particles that are collected, the more the temperature drops, hence the particles will fall faster as well.
The time left, the temperature, and the number of particles collected will be displayed on the screen. The arrow keys or the 'a' and 'd' keys can be used to move the tank, but the tank cannot leave the game screen.
The aim of the game is to collect as many greenhouse gas particles as possible within one minute. How many particles can you collect? Let's find out!

Personal Tracker

 Using the personal personalTracker feature of Eco-Solve, it is now possible to check your current environmental footprint by entering data into our personal personalTracker.
In addition, you can set your own sustainability goals and reduce your negative impact on the environment.
We value privacy at Eco-Solve, so your data is stored in a .txt file and is not accessible to anyone else (even us, for that matter).
